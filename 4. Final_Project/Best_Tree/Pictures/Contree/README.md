Contree Overview:

- Cladogram_posterior show this rooted contree file that has the posterior probability values and is using cladogram to have the branches equal.

- Colored_cladogram is similar to Cladogram_posterior in having posterior probability values and using cladogram, the difference is that this tree is colored according to the values of the posterior probability according to the legend.

- No_cladogram shows this contree without an posterior values and not using cladogram.

- These trees when compared to the trees created in R, Beauti-Beast, and NGPhylogeny, this could be considered on par with the best trees.
