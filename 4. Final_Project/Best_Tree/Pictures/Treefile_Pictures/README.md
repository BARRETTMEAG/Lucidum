Treefile Pictures Overview:

- Lucidum is a tree that combines the use of R and treefile. There are three different colors indicating if an taxa has lucidum (green), without lucidum (red), and if it is unknown (grey).

- No_cladogram_colored_post... is a rooted colored tree that has the posterior values and is colored according to that values, this tree does not use cladogram.

- SM_branch_length is a rooted tree that shows the branch length.

- Cladogram_node_colored is a rooted colored tree that has posterior values and is using cladogram.
