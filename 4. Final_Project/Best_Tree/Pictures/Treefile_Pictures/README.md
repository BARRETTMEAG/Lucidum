Treefile Pictures Overview:

- Color_posterior_branch, a rooted colored tree that has the branch length values and colored according to posterior values, this tree does not use cladogram.

- Lucidum, a tree that combines the use of R and treefile. There are three different colors indicating if an taxa has lucidum (green), without lucidum (red), and if it is unknown (grey).

- No_cladogram_colored_post..., a rooted colored tree with posterior values and colored according those values, this tree does not use cladogram.

- SM_branch_length, a rooted tree that shows branch length.

- Cladogram_node_colored, a rooted colored tree with posterior values and uses cladogram.
